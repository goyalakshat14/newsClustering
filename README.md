# newsClustering

1. Make a mongoDB database by the name of feeduction

2. Make 4 collection by the name of:
	1. users
	2. feedsUnread
	3. feedsRead
	4. feedLinks

3. start the python server:
	 python news_clustering.py

4. start the node.js server:
	 node feeduction.js

5. open 127.0.0.1:3000 in browser

6. enjoy!
